# Overview Dashboard - Final Metrics List

3. **Weekly Active Users (WAU)** - 7-day active user count

8. **Total Prompts** - Volume of all prompts
9. **Enhanced Prompts** - Volume of enhanced prompts

13. **Refine Rate** - % of prompts refined by users
14. **Avg Processing Time** - Speed performance
